# Agentic AI: From Chatbots to Autonomous Agents

This repository serves as a practical guide and code showcase for building Agentic AI systems. While traditional LLM implementations focus on passive text generation, this project explores the transition to autonomous agents that can reason, use tools, and execute multi-step tasks to achieve a specific goal.

🤖 **What makes it "Agentic"?**

In this repo, we move beyond simple "input-output" loops. The agents here demonstrate:

*   **Reasoning**: Using techniques like **ReAct** (Reason + Act) or **Chain-of-Thought** to plan steps.
*   **Tool Use**: The ability to interface with the real world (web search, calculators, database queries, or custom APIs).
*   **Memory**: Utilizing both short-term context and long-term vector storage to maintain state.
*   **Autonomy**: Defining a high-level goal and letting the agent determine the necessary path to completion.

🚀 **Getting Started**

1.   **Clone the repo**: git clone https://github.com/nguyenhv75/agentic-ai.git
1.   **Install dependencies**: pip install -r requirements.txt
2.   **Set your API keys**: Create a .env file with your provider keys.
2.   **Run the demo**: python main.py

**Note**: Agentic AI is an evolving field. The goal of this repository is to document the "Best Practices" for building reliable, steerable, and safe autonomous systems.
